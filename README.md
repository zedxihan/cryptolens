# CryptoLens

**Clean and minimal cryptocurrency monitoring and portfolio tracker.**

A sleek, distraction-free dashboard to track your crypto holdings and
monitor the market in real time --- built with simplicity and beautiful
UX in mind.

## 🚀 Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/zedxihan/cryptolens.git
cd cryptolens
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```
